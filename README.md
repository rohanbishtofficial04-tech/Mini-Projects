# Mini Projects
Disease Prediction System

Disease Prediction System is a python & Flask-based system that predicts multiple diseases using patient health parameters and lab values through machine learning models
using Logistic Regression and Random Forest


FretLab – Guitar Utility Application

A lightweight guitar utility application designed to help guitarists practice more effectively.
It combines essential tools like a tuner, metronome, and chord library into a single, easy-to-use interface.
The app focuses on simplicity and accuracy, allowing users to tune their guitar, maintain proper rhythm, and learn chords.
FretLab is ideal for beginners as well as intermediate players looking for a streamlined practice companion.

ToneForge – Frequency Generator

Built a real-time frequency generator web application using Flask and the Web Audio API, enabling precise frequency control, waveform selection, stereo panning, and volume adjustment.
Implemented a complete musical notes system (C0–E10) with searchable, scrollable UI and an intuitive, responsive interface using HTML, CSS, and JavaScript.
